TrekAura — Website

What is this?
- This is the website for TrekAura — a small travel company offering Himalayan treks and local experiences.

What you'll find here
- A homepage with booking and contact details.
- Pages describing tour packages and itineraries.
- A photo gallery with images from past treks.

How to view the site (very simple)
- Open the `index.html` file in your web browser (double-click the file or drag it into your browser window).
- If you want a quick local preview using a simple server (optional):

  python3 -m http.server 8000

  then open http://localhost:8000 in your browser.
